# 在线教育系统（不够精美版）

本项目为**前后端分离**的在线教育系统，适合作为计算机专业本科毕业设计演示项目。

## 一、项目简介

系统面向三类用户：
- 学生：浏览课程、选课学习、提交作业、参与课程问答等
- 教师：创建课程、维护章节资源、发布作业、回复问答、提交课程审核等
- 管理员：审核课程、管理用户等

## 二、技术栈

### 前端
- Vue 3
- TypeScript
- Vite
- Pinia
- Vue Router
- Element Plus
- Axios

### 后端
- Spring Boot 3
- Spring Security
- JWT
- MyBatis-Plus
- MySQL 8
- Lombok
