# 接口清单

## 1. 认证模块
- POST /api/auth/login 登录
- POST /api/auth/register 注册

## 2. 用户模块
- GET /api/users/me 获取当前用户信息
- PUT /api/users/me 修改当前用户信息
- GET /api/admin/users 管理员分页查询用户
- POST /api/admin/users 管理员新增/修改用户
- DELETE /api/admin/users/{id} 管理员删除用户

## 3. 课程模块
- GET /api/student/courses 学生查询课程大厅
- GET /api/student/courses/{id} 学生查看课程详情
- GET /api/teacher/courses 教师查询我的课程
- POST /api/teacher/courses 教师新增/编辑课程
- POST /api/teacher/courses/{id}/submit-audit 教师提交审核
- DELETE /api/teacher/courses/{id} 教师删除课程
- GET /api/admin/courses 管理员查询课程
- POST /api/admin/courses/{id}/audit 管理员审核课程

## 4. 分类模块
- GET /api/categories 查询课程分类

## 5. 选课模块
- POST /api/enrollments/{courseId} 学生选课

## 6. 章节模块
- GET /api/chapters?courseId=xx 查询课程章节
- POST /api/chapters 新增/修改章节
- DELETE /api/chapters/{id} 删除章节

## 7. 资源模块
- GET /api/resources?chapterId=xx 查询章节资源
- POST /api/resources 新增/修改资源
- DELETE /api/resources/{id} 删除资源

## 8. 作业模块
- GET /api/assignments?courseId=xx 查询作业
- POST /api/assignments 新增/修改作业
- DELETE /api/assignments/{id} 删除作业

## 9. 提交模块
- POST /api/submissions 学生提交作业
- GET /api/submissions?assignmentId=xx 教师查看提交
- POST /api/submissions/{id}/score 教师评分

## 10. 问答模块
- GET /api/questions?courseId=xx 查询问答
- POST /api/questions 学生提问
- POST /api/questions/{id}/answer 教师回复

## 11. 上传模块
- POST /api/files/upload 文件上传
