# 系统功能说明

## 1. 角色划分

本系统包含三类角色：学生、教师、管理员。

## 2. 学生端功能

1. 用户登录
2. 个人信息查看与修改
3. 课程大厅浏览
4. 课程搜索
5. 课程详情查看
6. 选课
7. 查看课程章节与学习资源
8. 在线提交作业
9. 发布课程问题
10. 查看教师回复

## 3. 教师端功能

1. 用户登录
2. 个人信息维护
3. 我的课程列表管理
4. 新建课程
5. 编辑课程
6. 删除课程
7. 提交课程审核
8. 课程章节管理
9. 课程资源管理
10. 作业发布与删除
11. 查看学生提交
12. 学生成绩评分
13. 回复课程问答

## 4. 管理员端功能

1. 管理员登录
2. 审核课程
3. 维护审核意见
4. 用户分页查询
5. 新增用户
6. 编辑用户
7. 删除用户

## 5. 系统特点

1. 前后端分离，便于开发维护
2. 使用 JWT 实现无状态认证
3. 使用角色权限控制接口访问
4. 使用统一结果对象规范接口返回
5. 页面风格简洁，适合毕业设计演示
